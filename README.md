📌 فكرة المشروع

🎯 الهدف من المشروع
تطبيق أساسيات HTML و CSS: لإنشاء واجهة مستخدم نظيفة.
التدرّب على JavaScript DOM Manipulation: كيفية التفاعل مع عناصر HTML وتعديل محتواها باستخدام JavaScript.
